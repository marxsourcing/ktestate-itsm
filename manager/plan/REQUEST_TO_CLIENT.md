## 1. IT 시스템/모듈 목록 (필수)

> **목적**: 요구사항 접수 시 시스템/모듈 선택 드롭다운 및 담당자 자동 배정에 활용

### 필요 데이터

| 항목 | 형식 | 예시 |
|------|------|------|
| 시스템 목록 (20개) | 엑셀/텍스트 | ONE, SAP ERP, 경영정보(BIS) 등 |
| 시스템별 모듈 목록 | 계층 구조 | ONE > 인사, ONE > 급여, SAP > 회계 등 |
| 시스템별 기본 담당자 | 이름/이메일 | 요구사항 자동 배정용 |
| 시스템 분류/그룹핑 | 선택 | 있다면 제공 |

### 예상 시스템 목록 (확인 필요)

```
 1. ONE (통합업무시스템)
 2. SAP ERP
 3. 경영정보(BIS)
 4. 호텔멤버십(HMS)
 5. 인재원(LIME)
 6. 그룹웨어
 7. 인사시스템
 8. 급여시스템
 9. EDMS
10. 포탈
... (총 20개)
```

**✔️ 요청 형태**: 엑셀 파일 또는 정리된 텍스트

---

## 2. 요구사항 분류 체계 (필수)

> **목적**: 요구사항 등록/관리 시 일관된 분류 기준 적용

### 필요 데이터

| 항목 | 기본값 (확정 필요) | 비고 |
|------|-------------------|------|
| 요구사항 유형 | 기능추가 / 기능개선 / 버그수정 / 기타 | 추가/수정 가능 |
| 우선순위 체계 | 긴급 / 높음 / 보통 / 낮음 | 추가/수정 가능 |
| 상태 체계 | 접수 / 검토중 / 진행중 / 완료 / 반려 / 보류 | 추가/수정 가능 |

### 확인 질문

- [ ] 위 기본값을 그대로 사용해도 되나요?
- [ ] 추가하거나 변경이 필요한 항목이 있나요?
- [ ] 내부에서 사용 중인 분류 체계 문서가 있나요?

**✔️ 요청 형태**: 확정된 분류값 목록 (이메일 회신 또는 문서)

---

## 3. 기존 요구사항 이력 데이터 (선택/AI 고도화용)

> **목적**: AI의 유사 요구사항 추천 및 RAG 기반 답변 품질 향상

### 필요 데이터

| 항목 | 상태 | 비고 |
|------|------|------|
| 데이터 제공 가능 여부 | ⬜ 확인 필요 | 있으면 AI 품질 향상 |
| 데이터 형식 | - | Excel / CSV / DB Export |
| 예상 데이터 건수 | - | 대략적인 수량 |
| 개인정보 포함 여부 | - | 마스킹 처리 필요 시 |

### 활용 방안

- **유사 요구사항 추천**: 과거 유사 건 참조하여 중복 방지
- **예상 처리 시간 안내**: 과거 처리 이력 기반 소요 시간 예측
- **자동 분류 정확도 향상**: 기존 데이터 학습을 통한 AI 분류 개선

### 우선순위

- POC 1차 완료 후 적용 예정 (Week 4 이후)
- 데이터 없이도 기본 기능 동작 가능
- 있으면 AI 기능 품질 향상에 큰 도움

**✔️ 요청 형태**: Excel/CSV 파일 (개인정보 마스킹 처리 후)

---

## 배포/이관 안내 (참고)

현재 개발 환경에서 구축 진행 중이며, **개발 완료 후 클라이언트가 원하는 환경으로 배포/이관** 예정입니다.

| 항목 | 현재 (개발 환경) | 이관 시 |
|------|-----------------|--------|
| DB | 개발용 Supabase | 클라이언트 환경 |
| 인증 | 이메일/비밀번호 | SSO 등 협의 |
| 배포 | 개발 서버 (Vercel) | 원하시는 환경 |
| AI | Gemini 3 Flash (개발 계정) | 클라이언트 계정 |

```
제목: [IT요구사항관리 POC] 기초 데이터 요청

안녕하세요,

AI 기반 IT 요구사항관리 시스템 POC 개발을 위해
아래 데이터를 요청드립니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[필수] 2주차까지 필요

1. IT 시스템/모듈 목록
   - 관리 대상 시스템 20개 공식 명칭
   - 시스템별 하위 모듈 목록
   - 시스템별 담당자 정보 (자동 배정용)

2. 요구사항 분류 체계 확정
   - 요구사항 유형: 기능추가/기능개선/버그수정/기타
   - 우선순위: 긴급/높음/보통/낮음
   - 상태: 접수/검토중/진행중/완료/반려/보류
   → 위 기본값 사용 가능 여부 또는 수정 사항

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[선택] 4주차 이후 활용 (AI 고도화용)

3. 기존 요구사항 이력 데이터
   - 과거 접수된 요구사항 이력 (있다면)
   - 형식: Excel/CSV
   - 개인정보 마스킹 처리 후 제공

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

* 인프라(DB, 인증, 배포환경)는 현재 개발 환경에서 구축 중이며,
  개발 완료 후 원하시는 환경으로 배포/이관해 드립니다.

회신 부탁드립니다.
감사합니다.
```
