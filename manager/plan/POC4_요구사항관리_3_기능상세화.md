# POC4: AI 기반 IT 요구사항관리 시스템 - 기능 상세화

---

## AI 챗봇 시스템 상세

### 자연어 기반 요구사항 접수

**화면/작업 구성**:
- 챗봇 인터페이스 화면: 채팅 형태 UI, 메시지 입력창, 전송 버튼
- 요구사항 입력 안내: 시작 메시지, 예시 안내 표시
- 첨부파일 업로드: 파일 첨부 버튼, 드래그앤드롭, 업로드 진행 표시
- AI 분류 결과 확인: AI가 분류한 시스템/모듈/유형 표시, 수정 옵션

**주요 처리**:
- 자연어 입력 수신
- LLM 기반 시스템/모듈 자동 분류
- 요구사항 유형 자동 판단 (기능추가/개선/버그수정/기타)
- 분류 결과 생성 및 표시

---

### AI 추가 질문 생성

**화면/작업 구성**:
- 추가 질문 표시: AI가 생성한 보완 질문 메시지
- 사용자 답변 입력: 메시지 입력 및 전송
- 답변 기반 요구사항 보완: 보완된 내용 미리보기

**주요 처리**:
- 요구사항 완성도 분석
- 부족한 정보에 대한 질문 자동 생성
- 답변 기반 요구사항 내용 보강

---

### 분류 결과 확인 및 수정

**화면/작업 구성**:
- 분류 결과 카드: 시스템, 모듈, 유형, 제목, 상세내용 표시
- 시스템 수정 드롭다운: 20개 시스템 선택 가능
- 모듈 수정 드롭다운: 선택된 시스템의 모듈 목록
- 유형 수정 라디오: 기능추가/개선/버그수정/기타
- 내용 수정 텍스트박스: 제목, 상세내용 편집
- 최종 제출 버튼: 요구사항 접수 완료

**주요 처리**:
- 분류 결과 표시
- 사용자 수정 내용 반영
- 최종 요구사항 데이터 저장
- 접수 완료 알림 생성

## 유사 요구사항 검색 시스템 (RAG) 상세

### 유사 요구사항 검색

**화면/작업 구성**:
- 유사 요구사항 목록: 검색된 유사 건 카드 목록 (제목, 상태, 일자)
- 유사도 표시: 각 건별 유사도 퍼센트
- 유사 건 상세 모달: 상세 내용, 처리 답변 확인

**주요 처리**:
- 입력된 요구사항 벡터 임베딩 생성
- 벡터 DB에서 유사 요구사항 검색
- 유사도 계산 및 정렬
- 검색 결과 표시

### 중복 탐지 및 알림

**화면/작업 구성**:
- 중복 경고 알림: "유사한 요청이 이미 있습니다" 메시지
- 기존 요청 링크: 해당 요청으로 이동 버튼
- 계속 진행 옵션: 그래도 접수하기 버튼

**주요 처리**:
- 유사도 임계값 이상 건 탐지
- 중복 가능성 판단
- 경고 메시지 표시
- 사용자 선택에 따른 처리 분기

### AI 챗봇 기반 요구사항 접수 (메인 화면)

**화면/작업 구성**:
- 메인 화면 레이아웃: 챗봇 영역 + 접수 폼 미리보기
- 접수 시작 안내: 환영 메시지, 사용 방법 안내
- 진행 상태 표시: 입력중 → 분류중 → 확인중 → 완료
- 접수 완료 화면: 접수 번호, 상태, 예상 처리 시간

**주요 처리**:
- 챗봇 세션 시작
- 단계별 진행 상태 관리
- 접수 완료 처리
- 접수 확인 알림

---

### 내 요청 목록

**화면/작업 구성**:
- 요청 목록 테이블: 접수번호, 제목, 시스템, 상태, 접수일
- 상태별 필터: 전체/접수/진행중/완료/반려 탭
- 검색 기능: 키워드 검색
- 정렬 옵션: 최신순/오래된순

**주요 처리**:
- 로그인 사용자의 요청 목록 조회
- 필터/검색 조건 적용
- 페이징 처리

---

### 요청 상세 조회

**화면/작업 구성**:
- 요청 상세 화면: 제목, 시스템, 모듈, 유형, 상세내용
- 첨부파일 목록: 파일명, 다운로드 버튼
- 처리 상태 타임라인: 접수 → 진행중 → 완료 단계별 표시
- 처리 담당자 정보: 담당자명, 연락처
- 답변 내용 표시: 처리 완료 시 답변 내용

**주요 처리**:
- 요청 상세 정보 조회
- 첨부파일 다운로드
- 처리 이력 조회

---

### 추가 정보 제공

**화면/작업 구성**:
- 추가 정보 요청 알림: 담당자가 요청한 추가 정보 표시
