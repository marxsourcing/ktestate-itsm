```mermaid
graph TD

    Login --> Role{역할 확인}
    
    Role -->|요청자| Chat["/chat<br/>AI 채팅"]
    Role -->|담당자| Workspace["/workspace<br/>AI 협업"]
```

## 핵심 화면 레이아웃

모든 주요 화면은 **좌측 목록 + 우측 AI 채팅** 레이아웃을 따릅니다.

```
┌─────────────────────────────────────────────────────────────┐
│ 헤더 (네비게이션, 알림, 사용자)                               │
├────────────────┬────────────────────────────────────────────┤
│                │                                            │
│  사이드바       │        메인 콘텐츠                          │
│  (목록)        │        (AI 채팅)                            │
│                │                                            │
│  ┌──────────┐ │   ┌──────────────────────────────────┐    │
│  │ + 새 대화 │ │   │                                  │    │
│  ├──────────┤ │   │  대화 히스토리                    │    │
│  │ 항목 1   │ │   │  (스크롤)                        │    │
│  │ 항목 2   │ │   │                                  │    │
│  │ ...      │ │   └──────────────────────────────────┘    │
│  └──────────┘ │   ┌──────────────────────────────────┐    │
│                │   │ 입력창                    [전송] │    │
│                │   └──────────────────────────────────┘    │
├────────────────┴────────────────────────────────────────────┤
│ 액션바 (컨텍스트에 따른 버튼들)                               │
└─────────────────────────────────────────────────────────────┘
```

## 사용자 시스템 (요청자) 상세 플로우

### 프로필/내 정보 플로우

```mermaid
flowchart TD
    A([헤더 사용자 영역 클릭])
    
    A --> B[드롭다운 메뉴 표시]
    B --> C{메뉴 선택}
    
    C -->|내 프로필| D["/profile" 페이지 이동]
    D --> E[프로필 정보 표시]
    
    E --> F{작업 선택}
    F -->|이름 수정| G[이름 입력]
    G --> H[저장]
    H --> E
    
    F -->|아바타 변경| I[이미지 선택/업로드]
    I --> J[이미지 저장]
    J --> E
    
    F -->|비밀번호 변경| K[현재 비밀번호 입력]
    K --> L[새 비밀번호 입력]
    L --> M{비밀번호 유효?}
    M -->|아니오| N[오류 메시지]
    N --> K
    M -->|예| O[비밀번호 변경 완료]
    O --> E
    
    F -->|알림 설정| P[알림 옵션 선택]
    P --> Q[설정 저장]
    Q --> E
    
    C -->|알림 설정| P
    
    C -->|다크모드 토글| R[테마 변경]
    R --> S([메뉴 닫힘])
    
    C -->|로그아웃| T[세션 종료]
    T --> U[로그인 페이지 이동]
    U --> V([로그아웃 완료])
```

### AI 채팅으로 요구사항 접수 플로우

```mermaid
flowchart TD
    A([사용자 진입])
    
    A --> B[로그인]
    B --> C{인증 성공?}
    C -->|아니오| B
    C -->|예| D["/chat" 채팅 화면]
    
    D --> E[좌측: 대화 목록<br/>우측: AI 채팅]
    E --> F["+ 새 대화" 클릭]
    F --> G[AI 환영 메시지]
    
    G --> H[자연어로 요구사항 설명]
    H --> I[AI 실시간 응답 & 분석]
    
    I --> J{추가 정보 필요?}
    J -->|예| K[AI 추가 질문]
    K --> L[사용자 답변]
    L --> I
    
    J -->|아니오| M[AI가 요구사항 카드 생성<br/>채팅 내 표시]
    
    M --> N{카드 내용 확인}
    N -->|수정 필요| O[카드에서 직접 수정<br/>시스템/모듈/유형/내용]
    O --> M
    
    N -->|OK| P{유사 요청 발견?}
    P -->|예| Q[유사 요청 표시<br/>계속/취소 선택]
    Q -->|계속| R
    Q -->|취소| G
    
    P -->|아니오| R["요구사항 확정" 버튼 클릭]
    R --> S[요구사항 보드로 이동]
    S --> T([완료])
```

## 처리담당자 시스템 상세 플로우

### AI 협업 요청 처리 플로우

```mermaid
flowchart TD
    A([담당자 로그인])
    
    A --> B["/workspace" 워크스페이스]
    B --> C[좌측: 배정된 요청 목록<br/>우측: AI 협업 채팅]
    
    C --> D[우선순위별 그룹 표시<br/>긴급/높음/보통/낮음]
    D --> E[요청 카드 클릭]
    
    E --> F[우측 채팅 영역 활성화]
    F --> G[AI가 자동 분석 시작]
    
    G --> H[AI: 유사 사례 검색 결과 표시]
    H --> I{유사 사례 참고?}
    I -->|예| J[유사 사례 상세 보기<br/>기존 답변 참고]
    I -->|아니오| K
    J --> K
    
    K --> L{추가 정보 필요?}
    L -->|예| M["채팅: 요청자에게 화면 캡처 요청해줘"]
    M --> N[AI가 추가 정보 요청 메시지 생성]
    N --> O[요청자에게 알림]
    O --> P[요청자 응답 대기]
    P --> K
    
    L -->|아니오| Q{답변 초안 필요?}
    Q -->|예| R["채팅: 답변 초안 작성해줘"]
    R --> S[AI가 답변 초안 생성]
    S --> T[담당자가 수정/보완]
    T --> U[최종 답변 확정]
    
    Q -->|아니오| U
    
    U --> V[하단 액션바에서 상태 변경]
    V --> W["처리 완료" 버튼 클릭]
    W --> X[요청자에게 완료 알림]
    X --> Y([처리 완료])
```
