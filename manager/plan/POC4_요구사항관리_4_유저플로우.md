# POC4: AI 기반 IT 요구사항관리 시스템 - 유저 플로우

---

## 전체 시스템 구조

```mermaid
graph TD
    Start([시작])
    
    Start --> User[사용자 시스템<br/>요청자]
    Start --> Handler[처리담당자 시스템<br/>IT팀]
    Start --> Admin[관리자 시스템<br/>IT팀장]
    Start --> AI[AI 시스템]
    Start --> Data[데이터/인프라]
    
    User --> U1[로그인]
    User --> U2[AI 챗봇 요구사항 접수]
    User --> U3[내 요청 목록]
    User --> U4[요청 상세 조회]
    User --> U5[추가 정보 제공]
    
    Handler --> H1[담당자 로그인]
    Handler --> H2[배정된 요청 목록]
    Handler --> H3[요청 처리]
    Handler --> H4[추가 정보 요청]
    
    Admin --> A1[관리자 로그인]
    Admin --> A2[대시보드]
    Admin --> A3[전체 요청 관리]
    Admin --> A4[담당자 배정]
    Admin --> A5[시스템/모듈 설정]
    
    AI --> AI1[자연어 분석/분류]
    AI --> AI2[유사 요구사항 검색]
    AI --> AI3[답변 초안 생성]
    
    Data --> D1[시스템/모듈 데이터]
    Data --> D2[벡터 DB]
    Data --> D3[전자결재 연동]
    Data --> D4[인프라 구축]
```

## 사용자 시스템 (요청자) 상세 플로우

### 요구사항 접수 플로우 (메인)

```mermaid
flowchart TD
    A([사용자 진입])
    
    A --> B[로그인 화면]
    B --> C{인증 성공?}
    C -->|아니오| B
    C -->|예| D[메인 화면<br/>AI 챗봇 인터페이스]
    
    D --> E[자연어로 요구사항 입력]
    E --> F[첨부파일 업로드<br/>선택]
    F --> G[AI 분석 중...]
    
    G --> H{추가 정보 필요?}
    H -->|예| I[AI 추가 질문 표시]
    I --> J[사용자 답변 입력]
    J --> G
    
    H -->|아니오| K[AI 분류 결과 표시<br/>시스템/모듈/유형]
    
    K --> L{분류 결과 확인}
    L -->|수정 필요| M[분류 결과 수정<br/>시스템/모듈/유형 변경]
    M --> K
    
    L -->|확인| N{유사 요구사항 있음?}
    N -->|예| O[유사 요구사항 목록 표시]
    O --> P{중복 여부 확인}
    P -->|중복| Q[기존 요청 상세 조회]
    P -->|신규| R[최종 제출]
    
    N -->|아니오| R
    
    R --> S[접수 완료 화면<br/>접수번호 표시]
    S --> T([완료])
```

### 내 요청 조회 플로우

```mermaid
flowchart TD
    A([로그인 완료])
    
    A --> B[내 요청 목록 화면]
    B --> C[상태별 필터 선택<br/>전체/접수/진행중/완료/반려]
    C --> D[검색/정렬 적용]
    D --> E[요청 목록 표시]
    
    E --> F[요청 항목 클릭]
    F --> G[요청 상세 화면]
    
    G --> H[처리 상태 타임라인 확인]
    G --> I[첨부파일 다운로드]
    G --> J[처리 답변 확인]
    
    G --> K{추가 정보 요청 있음?}
    K -->|예| L[추가 정보 입력 폼]
    L --> M[추가 정보 제출]
    M --> G
    
    K -->|아니오| N([조회 완료])
```

## 처리담당자 시스템 상세 플로우

### 요청 처리 플로우

```mermaid
flowchart TD
    A([담당자 로그인])
    
    A --> B[배정된 요청 목록]
    B --> C[필터/검색 적용<br/>상태/시스템/우선순위]
    C --> D[요청 목록 표시]
    
    D --> E[요청 항목 클릭]
    E --> F[요청 처리 화면]
    
    F --> G[요청 상세 정보 확인]
    F --> H[유사 사례 패널 확인<br/>AI 제공]
    F --> I[첨부파일 확인]
    
    G --> J{추가 정보 필요?}
    J -->|예| K[추가 정보 요청 버튼]
    K --> L[요청 내용 입력]
    L --> M[요청자에게 전송]
    M --> N[상태: 정보요청중]
    
    J -->|아니오| O[처리 진행]
    O --> P[상태 변경: 진행중]
    
    P --> Q[답변 작성]
    Q --> R{AI 초안 사용?}
    R -->|예| S[AI 답변 초안 생성 버튼]
    S --> T[AI 초안 표시]
    T --> U[초안 수정/편집]
    U --> V[답변 완성]
    
    R -->|아니오| V
    
    V --> W{전자결재 필요?}
    W -->|예| X[전자결재 연동 버튼]
    X --> Y[결재 요청 생성]
    Y --> Z[결재 완료 대기]
    Z --> AA[상태 변경: 완료]
    
    W -->|아니오| AA
    
    AA --> AB[요청자에게 알림]
    AB --> AC([처리 완료])
```

## AI 시스템 상세 플로우

### AI 분류 및 검색 플로우

```mermaid
flowchart TD
    A([사용자 입력 수신])
    
    A --> B[자연어 분석]
    B --> C[LLM 호출<br/>시스템/모듈/유형 분류]
    C --> D[분류 결과 생성]
    
    D --> E[요구사항 완성도 분석]
    E --> F{완성도 충분?}
    F -->|아니오| G[추가 질문 생성]
    G --> H[사용자에게 질문 전송]
    H --> I([추가 답변 대기])
    
    F -->|예| J[벡터 임베딩 생성]
    J --> K[유사 요구사항 검색<br/>벡터 DB]
    K --> L{유사도 높은 건 있음?}
    L -->|예| M[유사 요구사항 목록 생성]
    L -->|아니오| N[분류 결과만 반환]
    
    M --> O([결과 반환])
    N --> O
```
